Сайт автоматически формирует ссылки на страницы репозиториев, собирая их из отдельных частей. Соберите ссылку с помощью конкатенации и выведите на экран:

```text
https://github.com/hexlet/exercises-javascript
```

Каждый компонент URL — отдельная строка: протокол, домен, имя владельца, имя репозитория.
