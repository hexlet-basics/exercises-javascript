El sitio genera automáticamente enlaces a las páginas de los repositorios, ensamblándolos a partir de partes separadas. Ensambla el enlace usando concatenación e imprímelo en la pantalla:

```text
https://github.com/hexlet/exercises-javascript
```

Cada componente de la URL es una cadena separada: el protocolo, el dominio, el nombre del propietario y el nombre del repositorio.
