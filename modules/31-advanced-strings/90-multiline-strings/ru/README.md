Для многострочного текста обычно используют `\n`:

```javascript
console.log('Строка 1\nСтрока 2\nСтрока 3');
```

Но при большом количестве строк это неудобно.

## Шаблонные строки для многострочного текста

Шаблонные строки (в обратных кавычках) сохраняют все переносы строк буквально:

```javascript
const text = `Первая строка
Вторая строка
Третья строка`;

console.log(text);
// Первая строка
// Вторая строка
// Третья строка
```

Перенос в исходном коде становится `\n` в строке.

## Пример: email-письмо

```javascript
const email = `Уважаемый клиент!

Ваш заказ принят.
Спасибо за покупку!`;
```

Это удобнее, чем:

```javascript
const email = 'Уважаемый клиент!\n\nВаш заказ принят.\nСпасибо за покупку!';
```

## Важно

Отступы в коде войдут в строку. Если сделать отступ внутри шаблонной строки, он появится в тексте.

```javascript
const message = `Hello
  World`;  // строка содержит 2 пробела перед World
```
