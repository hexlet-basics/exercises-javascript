У некоторых функций есть **параметры со значением по умолчанию**. Если аргумент не передан, функция использует значение по умолчанию.

## Пример: Math.round()

```javascript
// Без второго аргумента — округляет до целого
console.log(Math.round(10.25));    // => 10

// Но в большинстве языков round() принимает второй аргумент — число знаков
// В JavaScript используем toFixed() для округления до N знаков
console.log((10.1234).toFixed(2)); // => '10.12'
console.log((10.1234).toFixed(0)); // => '10'
```

## Параметр по умолчанию

Параметр со значением по умолчанию — это необязательный параметр. Если не передать, функция использует заданное значение.

```javascript
// В документации необязательные параметры часто показаны в квадратных скобках:
// toFixed([digits])
// digits — количество знаков после запятой (по умолчанию 0)

console.log((3.14159).toFixed());  // => '3'   (digits = 0 по умолчанию)
console.log((3.14159).toFixed(2)); // => '3.14'
console.log((3.14159).toFixed(4)); // => '3.1416'
```

## Читайте документацию

Чтобы понять, какие параметры обязательны, а какие необязательны — читайте документацию (MDN). Квадратные скобки `[параметр]` в сигнатуре означают необязательный параметр.
