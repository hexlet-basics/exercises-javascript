В коде заданы данные об автомобильной поездке: расстояние, расход топлива, цена топлива и количество пассажиров.

Вычислите и выведите на экран три значения, каждое на отдельной строке:

1. Объём топлива в литрах, необходимый для поездки. Округлите до одного знака после запятой — `toFixed(1)`.
2. Общую стоимость поездки в рублях. Округлите до двух знаков — `toFixed(2)`.
3. Стоимость поездки для каждого пассажира. Округлите до целого, вызвав `toFixed()` без аргумента (значение по умолчанию — 0 знаков).

distance ──────┐

fuelConsumption ┘              ├──→ tripCost ──────┐
