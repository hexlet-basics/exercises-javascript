Функции — это именованные блоки кода, которые можно вызвать в любой момент. JavaScript предоставляет множество встроенных функций.

## Как вызвать функцию

```javascript
console.log('Hello!');
```

Здесь `console.log` — это функция. Вызов функции выглядит как её имя, за которым идут скобки с аргументами.

## Аргументы

Аргументы — это значения, которые передаются в функцию при вызове. Функция использует их для вычислений.

```javascript
console.log(1 + 2);        // один аргумент: число 3
console.log('Hello', '!'); // два аргумента
```

## Возвращаемое значение

Функции могут возвращать результат. Этот результат можно использовать в выражениях или сохранять в переменную.

```javascript
// Math.pow(base, exponent) — возводит base в степень exponent
console.log(Math.pow(2, 10)); // => 1024

// Результат можно сохранить
const result = Math.pow(2, 10);
console.log(result); // => 1024
```
