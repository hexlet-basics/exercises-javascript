Сайт проверяет, вместится ли реклама двух компаний-партнёров в одну строку. В коде определены их названия — посчитайте общее количество символов в обоих названиях с помощью функции `length()` и выведите на экран.

Функция уже подключена в заготовке:

```javascript
import { length } from 'hexlet-basics/string';
```
