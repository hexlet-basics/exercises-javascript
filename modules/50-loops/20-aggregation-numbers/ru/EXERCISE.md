Реализуйте функцию `calculateElectricityBill()`, которая принимает количество израсходованных киловатт-часов и возвращает сумму счёта за электричество.

Действует ступенчатый тариф: первые `100` кВт⋅ч стоят по `5` рублей, следующие `100` кВт⋅ч — по `7` рублей, а все кВт⋅ч сверх `200` — по `10` рублей.

Пройдите расход циклом и постепенно накопите итоговую сумму.

```javascript
calculateElectricityBill(80);  // => 400
calculateElectricityBill(150); // => 850
calculateElectricityBill(250); // => 1700
```
