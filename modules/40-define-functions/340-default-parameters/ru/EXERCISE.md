Напишите функцию `getHiddenCard(cardNumber, starsCount = 4)`, которая принимает номер карты (строку) и возвращает скрытый номер: звёздочки + последние 4 цифры.

```javascript
getHiddenCard('1234567890123456')    // => '****3456'
getHiddenCard('1234567890123456', 2) // => '**3456'
```

Используйте `'*'.repeat(starsCount)` и `.slice(-4)`.
