Реализуйте функцию `formatPrice()`. Она принимает число и возвращает его строковое представление, округлённое до двух знаков после запятой.

Используйте функцию `round()` из модуля `hexlet-basics/math` — это другой модуль того же пакета `hexlet-basics`, с которым вы работали в прошлом уроке. Импорт напишите самостоятельно.

```javascript
formatPrice(12.3456); // => '12.35'
formatPrice(2.5);     // => '2.50'
formatPrice(10);      // => '10.00'
```

## Подсказка

* В начале файла импортируйте функцию: `import { round } from 'hexlet-basics/math';`
* Второй аргумент `round()` — количество знаков после запятой
