Некоторые значения не меняются никогда: число π, фиксированный курс валют, лимит попыток входа. Такие значения называют **константами**.

## const в JavaScript

В JavaScript константы объявляются через `const`.

```javascript
const dollarsPerEuro = 1.25;
const maxLoginAttempts = 3;
```

Попытка изменить константу вызывает ошибку.

```javascript
const maxLoginAttempts = 3;
maxLoginAttempts = 5; // TypeError: Assignment to constant variable.
```

## const vs let

```javascript
let score = 0;       // будет меняться
const lives = 3;     // не меняется
```

Используйте `const` по умолчанию. Переходите на `let` только тогда, когда точно знаете, что значение будет переприсваиваться. Это упрощает чтение кода: увидев `const`, читатель сразу знает — значение не изменится.

## В Python

В Python отдельного ключевого слова для констант нет. Там принято писать имена заглавными буквами (`MAX_LOGIN_ATTEMPTS = 3`), но это только соглашение — ничего не мешает переприсвоить.
