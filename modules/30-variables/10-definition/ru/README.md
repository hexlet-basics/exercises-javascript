Переменные — один из базовых инструментов программирования. С их помощью программы запоминают и используют данные.

Рассмотрим пример.

```javascript
let greeting = 'Father!';
console.log(greeting); // => Father!
```

Мы **определили переменную** с именем `greeting` и присвоили ей значение `'Father!'`. Теперь вместо строки `'Father!'` можно использовать имя `greeting`.

```javascript
let greeting = 'Father!';
console.log(greeting); // => Father!
console.log(greeting); // => Father!
```

Значение переменной берётся в момент обращения к ней.

## let и const

В JavaScript переменные объявляются двумя способами:

- `let` — для значений, которые могут меняться.
- `const` — для значений, которые не будут меняться (константы).

```javascript
let currentUser = 'Alice';    // может поменяться позже
const maxAttempts = 3;        // никогда не изменится
```

Если не знаете, будет ли значение меняться — используйте `const`. Это хорошая практика.

## Правила именования

Имена переменных в JavaScript:

- Могут содержать буквы, цифры, `_` и `$`.
- Не могут начинаться с цифры.
- Чувствительны к регистру: `greeting` и `Greeting` — разные переменные.

Имена по стилю записываются в lowerCamelCase: `firstName`, `maxLoginAttempts`.
