Компьютеру всё равно, как называются переменные. Но код читают люди — и для них это имеет огромное значение.

Посмотрите на два примера.

```javascript
const x = 'John';
const abcd = 30;
```

vs

```javascript
const firstName = 'John';
const age = 30;
```

Первый вариант написан для компьютера, второй — для человека.

## Требования к именам

- Имя должно отражать содержимое, а не тип: `userName` лучше, чем `userString`.
- Используйте английский язык: `firstName`, а не `имя`.
- Не бойтесь длинных имён: `maxLoginAttempts` лучше, чем `mla`.

## Сложная задача

Как назвать переменную, которая хранит количество братьев и сестёр короля?

Какой из вариантов вы бы выбрали?

- `x` — непонятно ничего
- `count` — count чего?
- `siblingsCount` — понятно и точно ✓

Именование — одна из самых сложных задач в программировании. Со временем хорошие имена приходят сами собой.
