Переменная потому и называется переменной, что её значение можно **изменить**.

```javascript
let greeting = 'Father!';
console.log(greeting); // => Father!

greeting = 'Mother!';
console.log(greeting); // => Mother!
```

Обратите внимание: ключевое слово `let` пишется только при **первом** объявлении переменной. При повторном присваивании `let` уже не нужен.

## Зачем это нужно?

Переменные меняются в самых разных ситуациях:

- Пользователь обновляет своё имя в профиле.
- Игровой персонаж теряет или находит жизни.
- В процессе вычислений накапливается промежуточный результат.

## const не меняется

Если переменная объявлена через `const`, её нельзя переприсвоить.

```javascript
const greeting = 'Father!';
greeting = 'Mother!'; // TypeError: Assignment to constant variable.
```

Поэтому для значений, которые будут меняться, используйте `let`.
