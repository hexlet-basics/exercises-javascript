Статус заказа обновляется по мере его продвижения. В начале заказ находится «в пути», затем переходит в статус «доставлен».

В упражнении уже определена переменная `deliveryStatus` со значением `'в пути'`. Переопределите её значение на `'доставлен'` и выведите на экран.

Пример переопределения переменной:

```javascript
let someVar = 'старое значение';
someVar = 'новое значение';
console.log(someVar); // => новое значение
```
