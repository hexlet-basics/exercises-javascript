Кнопка в приложении работает как переключатель: каждое нажатие меняет состояние на противоположное. Реализуйте функцию `flipFlop(str)`: если строка равна `'flip'`, вернуть `'flop'`; в противном случае вернуть `'flip'`.

```javascript
flipFlop('flip'); // => 'flop'
flipFlop('flop'); // => 'flip'
```

Используйте тернарный оператор.
