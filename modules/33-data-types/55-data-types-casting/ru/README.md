JavaScript автоматически преобразует типы в некоторых ситуациях (неявное преобразование). Но иногда нужно преобразовать тип **явно** — когда точно знаете, что хотите получить.

## Функции преобразования

```javascript
Number('42');       // => 42      (строка → число)
Number('3.14');     // => 3.14
Number('hello');    // => NaN     (не число)

String(42);         // => '42'    (число → строка)
String(true);       // => 'true'

Boolean(0);         // => false
Boolean('');        // => false
Boolean('hello');   // => true
Boolean(1);         // => true
```

## Когда это нужно

Чаще всего при работе с пользовательским вводом (всё приходит как строка) или при формировании текстовых сообщений из чисел.

```javascript
const price = 100;
const message = 'Цена: ' + String(price) + ' руб.';
// Хотя в этом случае удобнее интерполяция: `Цена: ${price} руб.`
```

## Math.trunc() для целой части

Чтобы отбросить дробную часть числа, используют `Math.trunc()`:

```javascript
Math.trunc(36.6);  // => 36
Math.trunc(-36.6); // => -36
```

В отличие от `Math.floor()`, `Math.trunc()` просто отбрасывает дробную часть независимо от знака.
