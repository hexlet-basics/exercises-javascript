Напишите функцию `isInternationalPhone(phone)`, которая принимает строку с номером телефона и возвращает `true`, если номер начинается с символа `'+'`, и `false` в противном случае.

```javascript
isInternationalPhone('+7 (999) 456-78-90'); // => true
isInternationalPhone('8 (999) 456-78-90');  // => false
```
